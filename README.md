# CryptoHub Mobile - React Native

Aplicação mobile para iOS e Android que centraliza dados de múltiplas exchanges de criptomoedas.

## Recursos

- Visualização de patrimônio total consolidado
- Gráfico de desempenho dos últimos 7 dias
- Lista de exchanges conectadas com saldos individuais
- Navegação por tabs (Início, Exchanges, Carteira, Perfil)
- Design clean inspirado no app da B3

## Como executar

### Pré-requisitos

- Node.js instalado
- Expo CLI: `npm install -g expo-cli`

### Instalação

\`\`\`bash
npm install
\`\`\`

### Executar

\`\`\`bash
# Iniciar servidor de desenvolvimento
npm start

# Executar no Android
npm run android

# Executar no iOS
npm run ios
\`\`\`
